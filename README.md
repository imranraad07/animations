* Zoom Animations
* Card Animations
* Split Animations
* Shrink Animations
* In And Out Animations
* Swipe Left Animations
* Swipe Right Animations
* Slide Left Animations
* Slide Right Animations
* Slide Down Animations
* Slide Up Animations
* Fade Animations
* Windmill Animations
